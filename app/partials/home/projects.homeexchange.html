Projects

HomeExchange.com

Home Exchange is a house swapping website that allows travelers to swap their homes with other travelers in destinations they are wanting to travel to. Home Exchange has done over 300,000 online house exchanges and has 46,000+ current listings.

My Tasks

  • Core search functionality and algorithm

    I helped build the companies core components including their search engine. The Home Exchange search has a smart algorithm that returns results based on users who are looking to come to the searchers current area, it automatically priorities houses of similar value and many other variables. I created filtering functionality from a wide range of options and indexing methods.

  • Search frontend

    I created a seamless Google like search expierence with no page loads and functionality that pushes the searchers geo query to the URL in an SEO friendly format.

  • Localization

    This was very key for this project due to the users being spread out across the globe. I helped with functionality to programmatically translates all geo URL structure, geo searching and most of all the exchange process.
